# Template: Service Business
## Strategic Minds AI — APP FACTORY

### Purpose
Service business website template. Plumbing, HVAC, landscaping, cleaning, etc. Lead gen focused. Light theme option.

### Tech Stack
- **Framework:** Next.js 14+ App Router
- **Language:** TypeScript (strict mode)
- **Styling:** Tailwind CSS + custom design tokens
- **Database:** Supabase PostgreSQL + RLS
- **Auth:** Supabase Auth
- **Deploy:** Vercel
- **PWA:** Yes — installable on mobile

### Design System
```css
--color-black: #000000
--color-charcoal: #1C1B19
--color-gold: #C09030
--font-display: 'Bebas Neue'
--font-body: 'Inter'
```

5. Run: `npm install --legacy-peer-deps`
6. Copy `.env.example` to `.env.local`
7. Deploy to Vercel
